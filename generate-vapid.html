🔑 M8 — VAPID key generator

These keys let the M8 server send push notifications to your phone. They are generated locally in your browser and shown only here — copy them into Vercel.

Then, in Vercel → your M8 project → Settings → Environment Variables (Production):

  1. Add VAPID_PUBLIC_KEY = the public value above.
  2. Add VAPID_PRIVATE_KEY = the private value above.
  3. Add VAPID_SUBJECT = mailto:m.7oofy@gmail.com.
  4. Save, then redeploy (or it applies on the next deploy).
